# Borg

Borg is a local-first agentic platform for autonomous agents.

It is local, fast, always-on, designed for the channels you already use, and sandboxed by default.

Run your own agents and workflows, connected to real tools, and keep your data and runtime under your control.

## First Run Flow

1. Add an LLM provider.
2. Create a Behavior for your Actor
3. Start your Actor!
2. Create a port (for example Telegram or HTTP) and connect it to your actor.
4. Start chatting and iterating.
